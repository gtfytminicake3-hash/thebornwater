# Original Decode Reconstruction Plan

## Rule Lock
This project rebuilds the original game behavior from decoded evidence exclusively.
- Do not invent gameplay, UI, backend logic, frontend flow, balance values, or assets.
- If evidence is missing, report `UNKNOWN` or `MISSING_FROM_CURRENT_DECODE`.
- Prototype fallback is allowed only for technical testing, not for FidelityMode.
- All non-verified numeric logic and flows must be segregated and clearly marked with `PROTOTYPE_FALLBACK`.
